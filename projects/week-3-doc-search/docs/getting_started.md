# Getting Started with Zen HR

Welcome to Zen HR, the modern HR management platform designed to streamline your people operations.

## Creating Your Account

1. Visit **app.zenhr.com**
2. Click "Start Free Trial"
3. Enter your work email address
4. Choose authentication method:
   - Google SSO (recommended for Google Workspace users)
   - Microsoft SSO (for Microsoft 365 users)
   - Email and password

No credit card is required to start your trial.

## What's Included in the Free Trial

Your 14-day trial includes full access to Professional plan features:

- Unlimited employees
- All HR modules (time-off, directory, documents)
- Performance reviews
- API access
- Priority email support
- Custom workflows
- Advanced reporting

## Your First Steps

### Step 1: Complete Company Setup

After signing up, you'll be guided through initial setup:

1. Enter your company name and size
2. Upload your company logo
3. Set your fiscal year start date
4. Choose your default timezone

### Step 2: Add Your First Employees

Navigate to **People > Add Employee** to add team members:

1. Enter employee details (name, email, job title, department)
2. Set their start date
3. Choose whether to send an invite email
4. Assign a manager (optional)

Employees receive an email invitation to set up their profile and password.

### Step 3: Configure Time-Off Policies

Go to **Settings > Time Off** to set up leave policies:

- Create policies for PTO, sick leave, personal days
- Set accrual rules (annual grant or per-pay-period)
- Define carryover limits
- Assign policies to employees

### Step 4: Set Up Departments

Organize your company structure in **Settings > Departments**:

- Create departments and sub-departments
- Assign department heads
- Set up approval chains for time-off requests

## Importing Existing Data

If you're migrating from another system, Zen HR supports bulk import:

1. Go to **Settings > Data > Import**
2. Download our CSV template
3. Fill in your employee data
4. Upload the completed file
5. Review and confirm the import

Supported import formats: CSV, XLSX
