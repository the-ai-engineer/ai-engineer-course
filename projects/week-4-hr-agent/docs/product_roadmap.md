# Product Roadmap

See what we've shipped recently and what's coming next.

## Recent Releases

### Version 3.2 (December 2024)

**Performance Reviews 2.0**
- Customizable review templates
- 360-degree feedback support
- Goal tracking integration
- Review cycle automation

**Mobile App Improvements**
- Redesigned time-off request flow
- Push notifications for approvals
- Offline mode for viewing profiles
- Touch ID / Face ID support

**New Integration: ADP Workforce Now**
- Bidirectional employee sync
- Compensation data (read-only)
- Automatic time-off accrual sync

### Version 3.1 (October 2024)

**Custom Workflows**
- Build automated HR processes
- Trigger actions on events (new hire, termination, etc.)
- Send notifications and reminders
- Integrate with external systems via webhooks

**Advanced Reporting**
- 15 new pre-built reports
- Custom report builder
- Scheduled report delivery
- Export to PDF, CSV, XLSX

**Slack Integration**
- Slash commands for quick actions
- Channel notifications
- Time-off request approvals in Slack

### Version 3.0 (August 2024)

**New User Interface**
- Redesigned dashboard
- Improved navigation
- Dark mode support
- Accessibility improvements (WCAG 2.1 AA)

**API v1 Release**
- RESTful API for all features
- Webhook support
- OAuth 2.0 authentication
- Comprehensive documentation

**Document Management**
- Secure document storage
- E-signature support
- Document templates
- Audit trail

## Beta Program

Want early access to new features? Join our beta program:

1. Go to **Settings > Beta Features**
2. Click "Join Beta Program"
3. Agree to beta terms
4. Enable specific beta features

Beta features may be unstable and are not covered by our SLA.
